# 🧠 Mental Health Disease Prediction System

An AI-based system that predicts mental health conditions using supervised machine learning algorithms. The model analyzes survey responses related to stress, lifestyle, and personal factors to classify an individual's mental state.

---

## 📌 Project Overview

Mental health problems are often unnoticed until they become severe. This project uses machine learning to analyze survey data and predict mental health conditions, enabling early detection and support.

The system can classify individuals into emotional states such as:

- 😊 Happy  
- 😐 Sad  
- 😟 Stressed  
- 😞 Depressed  

---

## 🎯 Objectives

- Predict mental health condition using survey data  
- Compare multiple machine learning algorithms  
- Identify the most accurate model  
- Demonstrate AI application in healthcare  
- Provide early warning for mental health issues  

---

## 🧠 Machine Learning Algorithms Used

- Logistic Regression  
- Decision Tree  
- K-Nearest Neighbors (KNN)  
- Random Forest Classifier  

👉 Random Forest achieved the highest accuracy.

---

## 📊 Features Used for Prediction

The model analyzes factors such as:

- Work stress  
- Sleep habits  
- Social interaction  
- Family history  
- Workplace environment  
- Personal feelings  

---

## ⚙️ System Workflow

1. Load dataset  
2. Data preprocessing  
3. Encoding categorical features  
4. Train multiple ML models  
5. Compare accuracy  
6. Select best model  
7. Predict mental health condition
